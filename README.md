# ege

A repository for EGE (Единый государственный экзамен / Unified State Exam) preparation materials and resources.

## About

This project provides resources and materials to help students prepare for the EGE — the Russian national standardized exam taken by high school graduates.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/albert123-arch/ege.git
cd ege
```
